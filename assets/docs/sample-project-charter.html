2. Project Purpose

This charter formally authorises the FractalX research project with the objective of designing, implementing, and validating an automated static decomposition framework that converts modular Spring Boot monolithic applications into production-ready microservice platforms.

3. Scope

  • Design and implement the FractalX annotation model (@DecomposableModule, @DistributedSaga, @ServiceBoundary)
  • Build the AST analysis engine and code generation pipeline as a Maven plugin
  • Generate complete microservice platforms including API gateway, registry, admin, saga orchestrator, and logger
  • Implement NetScope - an automated gRPC communication fabric
  • Validate against a minimum of 3 real-world monolith scenarios
  • Publish to Maven Central and open-source under Apache 2.0
